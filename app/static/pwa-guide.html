米兰食品公司PWA安装指南

如何安装PWA应用

PWA (Progressive Web App) 是一种可以像原生应用一样安装到您设备上的网页应用。

注意： 您可能需要先清除浏览器缓存，以确保获取最新的PWA配置。

清除浏览器缓存

Chrome浏览器：

  1. 打开Chrome设置 (右上角三个点)
  2. 选择"隐私设置和安全性"
  3. 选择"清除浏览数据"
  4. 勾选"缓存的图片和文件"
  5. 点击"清除数据"

应用入口

米兰食品公司有三个独立的应用入口，每个都可以单独安装：

1. 客户端

点击访问客户端

适合餐馆用户下单使用。

2. 管理端

点击访问管理端

适合管理员使用。

3. 仓库端

点击访问仓库端

适合仓库员工使用。

安装步骤

在Chrome浏览器上安装 (Android)

  1. 打开相应的应用入口 (客户端、管理端或仓库端)
  2. 点击地址栏右侧的安装图标或菜单中的"安装应用"选项
  3. 在弹出的提示中点击"安装"

在Safari浏览器上安装 (iOS)

  1. 打开相应的应用入口 (客户端、管理端或仓库端)
  2. 点击底部的分享按钮
  3. 向下滚动并选择"添加到主屏幕"
  4. 点击右上角的"添加"

常见问题

我看不到安装选项

可能的原因：

  • 您已经安装了该应用
  • 您的浏览器不支持PWA
  • 网站配置有问题（请先清除浏览器缓存再试）

安装一个应用后，无法安装其他应用

解决方法：

  1. 清除浏览器缓存
  2. 卸载已安装的应用
  3. 重新访问相应入口
  4. 重新安装需要的应用
